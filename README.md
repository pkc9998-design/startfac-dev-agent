# \# 🧳 BagDrop — 인천공항 터미널 짐 보관 서비스

# 

# \## 프로젝트 목적

# 인천공항에는 주차 대행 서비스만 존재하여, 짐이 많은 여행객은

# 차를 맡기는 동안 짐을 직접 들고 다니거나 타인에게 차를 맡겨야 하는

# 불안과 불편함이 존재한다.

# 

# 본 서비스는 터미널 입구에서 고객이 주차하러 가는 동안 짐을 안전하게

# 보관해주고, 주차 완료 후 돌아오면 즉시 짐을 돌려주는

# \*\*공항 터미널 짐 보관 서비스\*\*를 제공한다.

# 

# \---

# 

# \## 서비스 개요

# 

# \### 출국 시

# 1\. 고객이 터미널 입구 도착

# 2\. 모바일앱에서 짐 보관 예약 및 결제

# 3\. 직원에게 짐 인계 후 주차장으로 이동

# 4\. 주차 완료 후 터미널로 복귀

# 5\. 직원에게 짐 수령

# 

# \### 귀국 시

# 1\. 고객이 터미널 도착 후 짐 인계

# 2\. 고객이 주차장으로 이동하여 차량 픽업

# 3\. 차량으로 터미널 입구 복귀

# 4\. 직원에게 짐 수령

# 

# \### 요금

# | 구분 | 요금 |

# |------|------|

# | 왕복 1회 (출국+귀국) | 5,000원 |

# 

# \---

# 

# \## 서비스 구성

# 

# \### 고객용 모바일웹 (FO)

# \- 예약: 출국일/시간, 귀국일/시간 선택

# \- 결제: 5,000원 왕복 결제

# \- 예약 확인 및 취소

# \- 예약 내역 조회

# 

# \### 운영자용 모바일웹 어드민 (BO)

# \- 예약 현황 조회 (날짜별/상태별)

# \- 결제 현황 확인

# \- 짐 인계/반환 상태 관리

# \- 고객 예약 상세 확인

#
